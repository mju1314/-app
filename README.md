# ExpenseTracker

一个面向个人自用的 Android 离线记账 App。当前版本已经完成从记账、查看、筛选、统计到导出和本地备份恢复的核心闭环。

## 当前能力

### 首页
- 查看今日支出和本月支出
- 查看最近记录
- 从首页进入记录详情

### 记一笔
- 新增一笔支出
- 选择分类、支付方式、备注
- 自动使用本地默认货币展示金额

### 明细
- 按时间倒序查看记录
- 按关键词搜索备注 / 分类 / 支付方式
- 按分类筛选
- 按时间范围筛选：全部、最近 7 天、本月
- 支持一键清空筛选条件
- 按天分组展示每日总额

### 统计
- 支持回看历史月份
- 查看所选月份总支出
- 查看分类占比
- 查看本月 Top 分类
- 查看本月日均支出
- 查看最近 7 天 / 30 天支出趋势

### 记录详情
- 查看完整记录信息
- 编辑记录
- 删除记录

### 设置
- 切换默认货币
- 导出 CSV
- 本地备份数据库和偏好设置
- 从本地备份恢复数据
- 清空全部记录

## 技术栈

- Kotlin
- Jetpack Compose
- Navigation Compose
- Room
- DataStore
- Hilt

## 数据与架构

- 本地优先：所有核心数据存储在本地，无需联网
- Room 负责交易记录、分类、支付方式的数据持久化
- DataStore 负责用户偏好设置
- ViewModel + Flow 负责页面状态组合与响应式更新
- Hilt 负责依赖注入

## 运行环境

- JDK 17
- Android SDK 34
- Gradle 8.6

## 本地运行

### 构建 Debug 包

```powershell
.\gradlew.bat assembleDebug
```

### 安装到设备或模拟器

```powershell
.\gradlew.bat installDebug
```

## 当前项目定位

这个项目的目标不是复杂财务系统，而是一个适合个人长期使用的轻量离线记账工具，重点是：

- 录入足够快
- 查询足够直接
- 统计足够清楚
- 数据可导出、可备份、可恢复
